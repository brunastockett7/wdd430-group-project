# Handcrafted Haven

This project was built using **Next.js** as part of the **WDD 430 course**.  
It focuses on creating a clean, responsive layout and practicing modern web development with components and routing.

The goal of this project was to:
- Build a responsive and user-friendly interface  
- Practice working with Next.js and TypeScript  
- Deploy a live application using Vercel  

---
